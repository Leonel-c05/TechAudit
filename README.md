# AuditTech

Aplicación móvil desarrollada en Android para la gestión de auditorías técnicas y laboratorios.  
El sistema permite registrar, visualizar y administrar información relacionada con auditorías dentro de la aplicación.

## Funcionalidades
- Registro de auditorías.
- Gestión de laboratorios.
- Visualización de detalles de auditoría.
- Edición y actualización de información.
- Consumo de datos mediante API.

## Tecnologías utilizadas
- Kotlin
- Android Studio
- Room Database
- Retrofit
- Arquitectura MVVM

## Estructura del proyecto
El proyecto está organizado en diferentes paquetes:

- **adapter**: Adaptadores para RecyclerView.
- **data**: Acceso a datos y base de datos (Room).
- **model**: Clases modelo de la aplicación.
- **network**: Consumo de API usando Retrofit.
- **ui**: Actividades y ViewModels de la interfaz de usuario.
